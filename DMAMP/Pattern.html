    Instantiate pattern class and acquire metadata for pattern from the appropriate pattern’s template directory. Will abort if the name given does not match a pattern directory giving a list of known patterns.

    # File lib/dmamp/pattern.rb, line 31
def initialize(name)
  @name = @pattern = name
  @templatedir = "#{PATTERNDIR}/#{@pattern}/templates"
  @knownpatterns = Dir.entries(PATTERNDIR).reject {|x| ["..","."].include?(x)}
  @manitemplates = Dir.entries(templatedir).reject {|x| ["..","."].include?(x)}
rescue Errno::ENOENT => e
  abort "Pattern #{@pattern} is not defined within the patterns directory #{PATTERNDIR}
  Known patterns are:
  #{@knownpatterns.join("\n")}"
end
